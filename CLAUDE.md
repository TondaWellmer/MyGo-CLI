# myGAUCLI — AI Chat Terminal

Stripped version of Pangea-CLI. Terminal + Chat only, no editors.

## Features
- Multi-Tab Terminal (PTY mit xterm.js)
- AI Chat (Free + Premium Models)
- Multi-Login (Google/Gemini, OpenCode)
- Session History + Work Sessions + Recovery
- Settings

## Tech
Electron, node-pty, xterm.js
